## 📋 Descripción

Portfolio web personal desarrollado con Node.js, Express y EJS, con un diseño moderno usando Tailwind CSS. Este proyecto muestra mis habilidades, proyectos y experiencia como desarrollador backend y estudiante de Ingeniería de Sistemas.

## ✨ Características

- **Diseño Responsivo**: Adaptado a dispositivos móviles, tabletas y escritorio
- **Animaciones Fluidas**: Implementadas con AOS (Animate On Scroll)
- **Secciones Completas**: Inicio, Proyectos, Sobre Mí y Contacto
- **Formulario de Contacto**: Integrado con Formspree para recibir mensajes
- **Métricas de Programación**: Integración con WakaTime
- **Cálculo Automático de Edad**: Se actualiza dinámicamente según mi fecha de nacimiento

## 🔧 Instalación y Uso

```bash
# Clonar el repositorio
git clone https://github.com/Bjohan23/Johan-becerra.git

# Cambiar al directorio del proyecto
cd Johan-becerra

# Instalar dependencias
npm install

# Ejecutar en modo desarrollo
npm run dev
```
